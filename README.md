# CSE702025-N06-Nhom-8
Hệ thống Quản lý thư viện là một phần mềm hỗ trợ các hoạt động quản lý sách, mượn trả tài liệu và người dùng trong thư viện một cách tự động và hiệu quả. Mục tiêu của hệ thống là giúp giảm thiểu thao tác thủ công, tiết kiệm thời gian, tăng tính chính xác và cải thiện trải nghiệm người dùng.
